# Medical Backend API (FastAPI + MongoDB)

Backend service for user management and medical document uploads.  
Designed to work with **Next.js frontend**

---

## 🚀 Tech Stack

- **FastAPI** – Async backend framework
- **MongoDB** – Metadata storage
- **Motor** – Async MongoDB driver
- **JWT** – Authentication
- **bcrypt** – Password hashing
- **Swagger / OpenAPI** – API documentation
